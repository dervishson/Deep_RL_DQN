# Deep_RL_DQN
This repo contains DQN, DQN with prioritized experience replay with binary heap tree, Double DQN, Dueling DQN by working with Lunar-Lander environment.
The models not fully optimized.

# Package specs
- gym 0.26.0
- torch 1.12.1
- python 3.9
